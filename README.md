# C-JSON-Parser

# Instalation

Debug object:
~~~ bash
cd c_json_parser
make
~~~

Release object:
~~~ bash
cd c_json_parser
make release
~~~

# Usage
Add `#include "<path_to_c_json_parser>/headers/c_json_parser.h"` at the top of the file.

Compile your project with `c_json_parser.o` object. This object is in c_json_parser/obj folder.
